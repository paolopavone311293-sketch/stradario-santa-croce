# 🚀 Stradario Santa Croce - Avvio Rapido

Webapp per cercare le 1.332 strade di Santa Croce con gestione (aggiungi/modifica/elimina).

---

## 📋 Checklist Setup (15 minuti)

- [ ] **Passo 1:** Leggi `CONFIG.txt` 
- [ ] **Passo 2:** Crea nuovo progetto Supabase → Copia credenziali in `index.html`
- [ ] **Passo 3:** Supabase → SQL Editor → Esegui `setup-database.sql`
- [ ] **Passo 4:** Supabase → Table Editor → Importa `strade.csv`
- [ ] **Passo 5 (opzionale):** Testa localmente aprendo `index.html` nel browser
- [ ] **Passo 6:** Pubblica su Netlify → Drag `index.html` → Online! 🎉

---

## ⚡ Quick Start

```
1. Leggi CONFIG.txt
2. Crea nuovo progetto Supabase
3. Copia credenziali in index.html
4. Esegui setup-database.sql in Supabase
5. Importa strade.csv in Supabase
6. Testa o pubblica su Netlify
```
